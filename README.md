# Projeto MetalFlex – Scripts SQL

Este repositório contém os scripts SQL do projeto da MetalFlex.  
Aqui estão os arquivos de criação das tabelas, inserção de dados e os comandos de SELECT, UPDATE e DELETE utilizados na atividade prática.

## Arquivos incluídos

- 01_create_tables.sql → Criação do banco e das tabelas.
- 02_inserts.sql → Inserção dos dados iniciais.
- 03_selects.sql → Consultas SQL com JOIN, WHERE, ORDER BY e LIMIT.
- 04_updates.sql → Atualizações de registros.
- 05_deletes.sql → Exclusão de registros usando WHERE.

## Como usar

1. Abra o MySQL Workbench.
2. Execute primeiro o arquivo de criação das tabelas.
3. Depois execute o arquivo de inserts.
4. Por fim, rode os selects, updates e deletes conforme necessário.

Esses arquivos fazem parte da atividade do mini-mundo desenvolvido na disciplina de Banco de Dados.
